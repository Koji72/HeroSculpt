# Ejemplo de Mapa de Desplazamiento
# Este archivo muestra cómo crear deformaciones basadas en texturas

# Estructura del mapa:
# - Blanco (255,255,255): Elevación máxima
# - Gris (128,128,128): Sin deformación  
# - Negro (0,0,0): Depresión máxima

# Patrones comunes:
# - Líneas del traje → Elevaciones sutiles
# - Costuras → Depresiones finas
# - Bordes → Elevaciones pronunciadas
# - Pliegues → Variaciones de altura
