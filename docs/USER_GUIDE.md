# 🦸‍♂️ Guía de Usuario - Superhero 3D Customizer

¡Bienvenido al **Superhero 3D Customizer**! Esta guía te ayudará a crear tu superhéroe personalizado paso a paso.

## 🚀 Primeros Pasos

### **Acceso a la Aplicación**
1. Abre tu navegador web
2. Ve a la URL de la aplicación
3. La aplicación se cargará automáticamente con un personaje base

### **Requisitos del Navegador**
- **Navegador moderno**: Chrome, Firefox, Safari, Edge (últimas versiones)
- **Soporte WebGL**: Necesario para renderizado 3D
- **Conexión a internet**: Para cargar modelos y recursos

## 🎮 Controles del Visor 3D

### **Rotación del Modelo**
- **Mouse**: Click izquierdo + arrastrar para rotar horizontalmente
- **Touch**: Deslizar con un dedo para rotar
- **Limitación**: Solo rotación horizontal (eje Y) para vista consistente

### **Zoom**
- **Mouse**: Scroll para acercar/alejar
- **Touch**: Pellizcar para hacer zoom
- **Límites**: Zoom automático para evitar problemas de visualización

### **Posicionamiento**
- **Centrado automático**: El modelo se centra automáticamente
- **Ajuste de vista**: La cámara se adapta al tamaño del modelo

## 🎨 Personalización del Personaje

### **Acceso al Selector de Partes**
1. **Botón "Personalizar"**: Click en el botón principal
2. **Modal overlay**: Se abre sin bloquear el visor 3D
3. **Navegación**: Usa las categorías en la parte superior

### **Categorías Disponibles**

#### **🏗️ Base**
- Modelo base del personaje
- Define la estructura fundamental
- Cambios afectan todo el personaje

#### **💪 Torso**
- Partes del pecho y abdomen
- Incluye trajes y armaduras
- Compatible con diferentes tipos de manos

#### **🦵 Piernas**
- Partes inferiores del personaje
- Botas y calzado especializado
- Diferentes estilos disponibles

#### **✋ Manos**
- Guantes y armas especializadas
- Tipos: abiertas, cerradas, armas
- Se adaptan automáticamente al torso

#### **🎭 Cabeza**
- Cascos y máscaras
- Diferentes estilos de superhéroe
- Compatible con capas

#### **🦇 Capa**
- Capas y accesorios de espalda
- Diferentes longitudes y estilos
- Se adapta al torso seleccionado

#### **👢 Botas**
- Calzado especializado
- Diferentes alturas y estilos
- Compatible con piernas

#### **🔧 Accesorios**
- Cinturones, símbolos, bolsas
- Detalles adicionales
- Personalización extra

## 🚨 Solución de Problemas

### **Problemas Comunes**

#### **El modelo no se carga**
- Verifica tu conexión a internet
- Recarga la página
- Limpia el caché del navegador

#### **Controles no responden**
- Asegúrate de que el visor esté enfocado
- Verifica que no haya otros elementos seleccionados
- Intenta hacer click en el visor

#### **Partes no se aplican**
- Verifica la compatibilidad de las partes
- Usa el botón "Aplicar" después de la selección
- Revisa si hay errores en la consola

#### **Rendimiento lento**
- Cierra otras pestañas del navegador
- Verifica que tu dispositivo tenga suficiente memoria
- Espera a que se complete la carga inicial

### **Optimización del Navegador**
- **Actualiza el navegador**: Usa la última versión
- **Habilita WebGL**: Asegúrate de que esté habilitado
- **Limpia caché**: Limpia regularmente el caché del navegador
